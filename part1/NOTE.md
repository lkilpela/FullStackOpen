## React 

1. **Install Node.js library And Create Vite**
```
# npm 6.x (outdated, but still used by some):
npm create vite@latest introdemo --template react

# npm 7+, extra double-dash is needed:
npm create vite@latest introdemo -- --template react
```
2. **Run the app**
```
cd introdemo
npm install
npm run dev
```
